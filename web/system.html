Exercise Sheet System - Full Deployment

System Overview

A complete rehabilitation exercise sheet generation system for elderly patients with cognitive and vision impairment. Produces printable A4 landscape cards in multiple formats: quadrant layouts, single large cards, and icon-only reference sheets.

1. System Requirements

  • Python 3.7+
  • ReportLab library: pip install reportlab
  • A4 landscape printer (recommended)

2. Installation

  1. Download exercise-sheet script
  2. Place in your working directory
  3. Make executable: chmod +x exercise-sheet
  4. Run generator in browser or use CLI directly

3. Available Exercises

★ Hand (Blue)
  • Stress Ball Squeeze
  • Finger Pinches
  • Water Bottle Squeeze
● Shoulder (Orange)
  • Gentle Arm Lift
  • Clasped Hands
  • Shoulder Circles
■ Arm (Green)
  • Nose Touch
  • Elbow Bend
  • Water Bottle Hold
▲ Leg (Purple)
  • Knee to Chest Lift
  • Seated Leg Raise
  • Leg Up with Toes

4. Output Formats

4.1 Quadrant Layout (Default)

4 exercise cards on a single A4 landscape page. Best for quick reference during sessions.

4.2 Single Card (2x Scaled)

1 card per A4 page, scaled 2x for visibility. 4 pages total. Ideal for patients with low vision.

4.3 Icon Sheets

Large icons and numbers only. Use for patient identification or labeling exercises.

5. Design Features

  • Colorblind-safe palette (Okabe-Ito): Blue, Orange, Green, Purple
  • Large text (24pt steps, 22pt headings) for vision impairment
  • Shape + number identification (★1, ●2, ■3, ▲4) for cognitive impairment
  • Goal box (top-right) for tracking daily completion
  • Color-matched text for exercise type and goal instructions
  • Laminate-friendly design for whiteboard marker use

6. CLI Usage Examples

# Default: 4 cards on 1 page
python3 exercise-sheet -1 hand_0 -2 shoulder_1 -3 arm_2 -4 leg_0
# Single cards (2x scaled), 4 pages
python3 exercise-sheet -1 hand_0 -2 shoulder_1 -3 arm_2 -4 leg_0 --layout single
# Custom output filename
python3 exercise-sheet -1 hand_1 -2 shoulder_0 -3 arm_1 -4 leg_2 -o patient-john.pdf

7. Patient-Specific Customization

  • Use the web generator to select exercises per patient
  • Admin mode allows adding custom exercises
  • Generate separate PDFs for each patient with their preferred exercises
  • Print and laminate for durability (supports whiteboard marker on laminated surface)
